# Security Policy

## Reporting a Vulnerability

The Limetest team takes security issues seriously. We appreciate your efforts to responsibly disclose your findings.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them directly to us by emailing [argo.mohrad@gmail.com](mailto:argo.mohrad@gmail.com).

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any ideas for mitigation

We'll acknowledge receipt of your report within 3 business days and will send a more detailed response indicating next steps.

Thank you for helping keep Limetest and our users safe!
